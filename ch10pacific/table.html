Term Acronym/Initialism Definition
Internet the interconnected network of computer networks
Hypertext Transfer Protocol HTTP a protocol used to communicate between the client computer and the web server, a set of rules for exchanging files such as text, graphic images, sound, video, and other multimedia files on the Web
Hypertext Markup Language HTML a text based language to format documents
World Wide Web WWW the graphical user interface providing access to information stored on web servers connected to the Internet
World Wide Web Consortium W3C a consortium that takes a proactive role in developing recommendations and prototype technologies related to the Web
Americans with Disabilities Act ADA A federal civil rights law that prohibits discrimination against people with disabilities
World Intelllectual Property Organization WIPO an organization dedicated to protecting intellectual property rights internationally
Client Computer work stations used by individuals such as a PC on a desk
Server Recieves requests from clients for resources such as a file
Client/Server Computers joined by a network, relationship between two computer programs the client requests some type of service from the server the server fulfils the request and transmits the results to the client over a network
Multipurpose Internet Mail Extensions MIME a standard for the exchange of multimedia documents among different computer systems, provides for the exchange of seven different media types on the Internet: audio, video, image, application, message, multipart, and text
Simple Mail Tansfer Protocol SMTP a protocol used to send e-mails to others
Internet Message Access Protocol IMAP a protocol used to recieve e-mails from others
File Transfer Protocol FTP a set of rules that allows files to be exchanged between computers on the Internet, commonly used to transfer web page files from computers to web servers
Transmission Control Protocol TCP the official communication protocol of the Internet, ensures the integrity of network communication
Internet Protocol IP set of rules that controls how data are sent between computers on the Internet, it routes a packet to the correct destination address
Domain Name System DNS associates IP addresses with the text-based URLs and domain names that are typed into a browser address box
Uniform Resource Locator URL a type of Uniform Resource Indentifier that represents the network location of a resource such as a web page, a graphic file, or an MP3 file; consists of the protocol, the domain name, and a heirarchical location of the file on the web server
Domain Name locates an organization or other entity on the Internet
Fully Qualified Domain Name The combination of host or subdomain, second-level domain, and top-level domain name
Top-level Domain TLD identifies the rightmost part of the domain name
Markup Language sets of directions that tell web browser software (and other user agent software that retrieves and renders web content) how to display and manage a web document
Element each individual markup code
Tag each individual markup code
Attribute modification of a tag that further describes its purpose
Extensible Hyper Text Markup Language XHTML uses tags and attributes of HTML4 along with the more rigorous syntax of XML
Hypertext Markup Language 5 HTML5 the successor to HTML and replaces XHTML; incorporates features of both HTML and XHTML, adds new elements, provides new functionality such as form edits and native video, and is designed to be backwards compatible
Document Type Definition DTD identifies the version of HTML in the document
Document Type DOCTYPE the DTD statement; is the first line of a web page document
